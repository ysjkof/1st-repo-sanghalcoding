비교 연산자와 블리언(Comparison operators & Boolean

이퀄(===)이라는 비교연산자는 Lt 항과 Rt 항의 관계에 따라서 true나 false, 둘 중 하나 값을 만들어내는 연산자다.
트루와 폴스라는 두 가지 값을 묶어서 블리언이라고 부른다.
Boolean은 True와 False 두 가지 값으로 된 데이터다.
document.write(1---1); 이때는 true라는 값이 된다.
document.write(1---2); 이때는 false라는 값이 된다.

if뒤에 갈로에는 불리언이 온다. if뒤에 불리언이 true면 바로 뒤 코드를 실행하고 false면 else안에 코드를 실행한다. 꺽쇠라는 문법이기 때문에 <를 쓴다.

리팩토링. 중복의 제거

중복되는 코드를 제거하고 최적화 하는 것.

input id="night_day" type="button" value="night" onclick="
if(document.querySelector('#night_day').value === 'night'){
document.querySelector('body').style.backgroundColor='black';
id셀렉터를 지우고 this를 사용해서 길이를 줄이고 같은 버튼을 여러개 배치할 수 있다.

input type="button" value="night" onclick="
if(this.value === 'night'){
document.querySelector('body').style.backgroundColor='black';
document.querySelector('body').style.color='white';
중복줄이기1-this
this는 이 이벤트가 소속된 태그를 가리키도록 약속돼 있음. 독립된 함수를 만들게 되면 이 this는 input버튼이 아니고 전역개체를 가리킨다. 웹브라우저에서는 윈도우가됨. 이해불가. 넘어가.
